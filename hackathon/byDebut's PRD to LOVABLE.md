# 2. CORE CONCEPT

byDebut transforms books into evolving narrative ecosystems where readers become creators.

Users can:

* select a book,
* modify one central variable,
* generate alternative realities,
* publish narrative variations,
* fork other users’ worlds,
* and participate in AI-powered collaborative storytelling.

The system uses AI agents to operate key business workflows automatically.
