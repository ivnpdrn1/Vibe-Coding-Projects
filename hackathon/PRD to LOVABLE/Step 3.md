# STEP 3 — Author Universe Mode (LAS GARGANTAS DEL DIABLO)

Now build the complete Author Universe Mode using:

LAS GARGANTAS DEL DIABLO

IMPORTANT:
This mode must feel fundamentally different from Public Domain Mode.

Public Domain Mode is about:

* reinterpretation,
* transformation of cultural classics,
* and exploration of inherited narratives.

Author Universe Mode is about:

* expansion of original IP,
* evolution of a living narrative universe,
* sequel discovery,
* emotional escalation,
* geopolitical branching,
* and long-term franchise potential.

The experience must feel:

* emotionally deeper,
* strategically larger,
* and more dangerous.

This mode should make judges understand that byDebut is not only for public domain reinvention.

It is also:

* a narrative infrastructure for original creators,
* an IP evolution platform,
* and a franchise exploration engine.

5. Discover the best audience for each fork.
6. Recommend the strongest expansion path.

IMPORTANT:
The tone must feel:

* cinematic,
* emotionally sophisticated,
* geopolitically intelligent,
* and prestige-level.
